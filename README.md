# 📖 Linux-Dark-Grimoire

**La biblia definitiva del Linux malévolo y defensivo.**  
Desde `ls` hasta rootkits, hardening forense y pentesting. Esto no es solo "aprender Linux", es pensar como un hacker y defender como un analista de élite. 🐧☠️

---

## ⚠️ Código de Honor

> Este conocimiento es para **pruebas de seguridad autorizadas**, laboratorios propios y CTFs.  
> Usarlo sin permiso es ilegal y te hará merecedor de una demanda o una putiza cósmica.  
> **Saber es poder, pero con poder viene responsabilidad, carnal.**

## 📂 Estructura del Grimorio

| Carpeta | ¿Qué encontrarás? |
|---------|-------------------|
| [`00-Scripts/`](00-Scripts/) | Scripts en Bash/Python para automatizar el caos |

| [`06-Persistence/`](06-Persistence/) | Backdoors, cronjobs ocultos, servicios maliciosos |
| [`07-Stealth/`](07-Stealth/) | Anti-forensics, borrado de logs, rootkits básicos |
| [`Resources/`](Resources/) | Cheatsheets, libros, distros recomendadas |
